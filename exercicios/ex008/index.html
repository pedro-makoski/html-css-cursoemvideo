Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a incorreta tag <b> que não é semântica

Porém agora corretamente temos um termo em destaque, usando a tag <strong> que agora sim é semântica

Itálico / Enfase

Nesta frase, temos um termo em Itálico usando a tag não semântica <i>

Agora temos nesta frase um termo em ênfase usando a tag <em> que é semântica

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

E no outro paragrafo, temos outro texto marcado no final

Texto grande e pequeno

Estamos criando um texto grande usando a tag <big> - que já esta obsoleta(não é semântica) e um texto pequeno usando a tag <small> que é semântica e ainda não está obsoleta, nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído com a tag <del> para indicar que ele deve ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido usando a tag <ins> para dar uma ênfase e indicar que ele foi adicionado depois

Existe também o texto sublinhado com a tag <u> (não semantica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 usando a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O usando a tag <sub>
